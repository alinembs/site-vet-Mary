# site-vet-Mary

## Cálculo de Medicamentos

Este projeto é uma página web simples para calcular a **dose (em mL)** de um medicamento para animais, a partir de:

- **Dose desejada** (mg/kg)
- **Peso** (kg)
- **Concentração do fármaco** (mg/mL)

A conta utilizada é:

\[ \text{Dose (mL)} = \frac{\text{Dose (mg/kg)} \times \text{Peso (kg)}}{\text{Concentração (mg/mL)}} \]

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Preencha os campos:
   - `Dose (mg/Kg)`
   - `Peso (Kg)`
   - `Concentração do fármaco (mg/ml)`
3. Clique em **Calcular**.
4. Para limpar os campos e o resultado, clique em **Limpar**.

## Arquivos do projeto

- `index.html` — estrutura da página
- `style.css` — estilos
- `script.js` — lógica do cálculo e controle dos campos
- `imagem-fundo-02.png` — imagem de fundo
- `WhatsApp Image 2025-04-27 at 18.51.21.jpeg` — imagem adicional (se usada na página)

## Observações

- O cálculo é exibido com **2 casas decimais**.
- Se algum campo estiver vazio, o sistema exibe uma mensagem pedindo preenchimento correto.
